Mode d'emploi du Cabinet V.A.L.E.U.R©

1. Connexion
Mot de passe « soleil ».

2. Dashboard
Post-it pour notes rapides, agenda, patients prioritaires (épinglés via l'étoile ☆).

3. Gestion des patients
Création, modification, suppression. Fiche patient complète avec scores, portrait, historique, objectifs, documents, journal de séances.

4. Agenda
Ajoutez des rendez-vous avec date, patient, notes.

5. Bibliothèque
Fiches cliniques. Pour ajouter une fiche : déposez le fichier HTML dans le dossier /fiches/ sur IONOS, puis ajoutez une entrée dans catalogue.json (code, cat, titre, fichier, dispo: true).

6. Sauvegarde et transfert entre appareils
Utilisez « 💾 Sauvegarder (JSON) » pour télécharger un fichier contenant toutes vos données. Pour transférer sur un autre ordinateur/tablette, enregistrez ce fichier sur iCloud Drive (ou un autre cloud), puis sur le nouvel appareil, ouvrez le cabinet et utilisez « 📂 Restaurer une sauvegarde ». Faites un export régulier !

7. Synchronisation cloud (Vercel)
Vos données sont automatiquement synchronisées avec votre API Vercel. Vous pouvez les retrouver sur n'importe quel appareil connecté au même compte. En bas à droite, un indicateur vous indique l'état de la synchronisation.

8. Déconnexion
Cliquez sur « Déconnexion » pour verrouiller l'accès au cabinet.
